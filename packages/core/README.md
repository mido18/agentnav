# @agentnav/core

Core schemas, generators, safety normalization, warnings, and readiness scoring for AgentNav.

AgentNav is the agent-readiness layer for React and Next.js. It helps websites expose `/llms.txt`, `/.well-known/agent.json`, `/.well-known/actions.json`, JSON-LD, safe action metadata, and an agent-readiness score.

See the main README: https://github.com/mido18/agentnav#readme
