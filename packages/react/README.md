# @agentnav/react

React components for making websites readable, navigable, actionable, and safer for AI agents.

Use `AgentNavProvider`, `AgentEntity`, `AgentField`, `AgentAction`, and `AgentForm` to add `data-agent-*` attributes and safety metadata to React apps.

AgentNav is the agent-readiness layer for React and Next.js.

See the main README: https://github.com/mido18/agentnav#readme
