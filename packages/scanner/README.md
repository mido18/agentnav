# @agentnav/scanner

Browser runtime scanner for AgentNav. It inspects the current page for metadata, navigation, JSON-LD, entities, actions, forms, safety rules, and warnings, then returns an `AgentPage` with a readiness score.

AgentNav is the agent-readiness layer for React and Next.js.

See the main README: https://github.com/mido18/agentnav#readme
