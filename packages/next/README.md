# @agentnav/next

Next.js route helpers for publishing AgentNav metadata files:

- `/llms.txt`
- `/.well-known/agent.json`
- `/.well-known/actions.json`
- Schema.org JSON-LD

AgentNav is the agent-readiness layer for React and Next.js.

See the main README: https://github.com/mido18/agentnav#readme
